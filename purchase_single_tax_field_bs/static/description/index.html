Purchase Single Tax Field

This module will help you to update tax through a single field with an update tax button.You do not need to fill the tax when adding a new product to the purchase order, it will be calculated automatically.

Overview

This Module will help you to automate tax update process. To do so need to click update tax button when creating a purchase order for the first time only.

Key Features

User can update tax through a single tax field with an update tax button

When adding a new product to the purchase order tax will be calculated automatically
